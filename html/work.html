PROJECTS

As an undergrad, I also got several opportunities to expand my network by engaging in different clubs and activities. These clubs and activities were mostly at my college, however, some activities were outside of my college and hosted by top companies. Listed below are the main clubs and activities that I was part of:

User-Level Thread Library

This project involved designing and implementing a user-level thread library which supports thread creation, thread scheduling via context-switching, thread synchronization using Semaphores, and Inter-Thread Communication(ITC) mechanism using Message Passing and Mailbox functionality. It utilizes high level C language to achieve all of the above given functionality.

Codebase of this project can be accessed using the link: User-Level Thread Library


User-Level Unix Shell Implementation

This project involved learning how a Unix shell works, writing a simple shell, creating processes, handling signals, using Unix system calls and C library function calls, and becoming a better programmer in general. The user-level shell implements built-in commands like cd, pwd, which, where, kill, watchuser etc., external commands like ls, grep, vim, rm etc. using exec() system call of Unix, Inter-Process Communication(IPC) mechanism using Piping functionality, and File Redirection mechanism. This project, like the first one, also utilizes high level C language for the implementation.

Codebase of this project can be accessed using the link: User-Level Unix Shell


Garden Design Desktop Application

This is, by far, my one of the favorite projects. In this desktop application, user can create his/her own garden with some cool features in-place. Those features include but not limited to: drag-n-drop event for plants, serializing/deserializing for saving/loading garden, customizable background, and different garden menus for end-user and application interaction. This application utilizes Model-View-Controller(MVC) Design Pattern and Object-Oriented-Programming(OOP) concepts in Java programming language.

Codebase of this application can be accessed using the link: My Cool Garden App
